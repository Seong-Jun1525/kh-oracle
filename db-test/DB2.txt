MEMBERS 테이블 생성 CREATE문의 요구사항은 MEMBER_ID에 기본키 설정을해야 하고 MEMBER_PWD에 NOT NULL 제약 조건을 설정하고 MEMBER_NAME은 고정이 아닌 가변길이로 설정해야한다.​
CREATE TABLE MEMBERS(
MEMBER_ID VARCHAR2(20),
MEMBER_PWD VARCHAR2(20) NOT NULL,
MEMBER_NAME CHAR(30),
MEMBER_AGE NUMBER,
MEMBER_EMAIL VARCHAR2(50),
PRIMARY KEY(MEMBER_ID)
);
MEMBER_NAME CHAR(30) => 고정 길이이므로 잘못되었다.

​BOARDS 테이블 생성 CREATE문의 요구사항은 BOARD_NO 컬럼에 기본키 설정을 해야하고 BOARD_WRITER 컬럼에 MEMBERS 테이블의 MEMBER_ID를 참조하는 외래키를 설정해야하고 BOARD_REG_DATE는 현재시간으로 기본값을 설정해야한다.
CREATE TABLE BOARDS(
BOARD_NO NUMBER PRIMARY KEY,
BOARD_TITLE VARCHAR2(100),
BOARD_CONTENT VARCHAR2(4000),
BOARD_WRITER VARCHAR2(20) FOREIGN KEY MEMBERS(MEMBER_ID),
BOARD_REG_DATE DATE DEFAULT '2023-03-22'
);

BOARD_WRITER VARCHAR2(20) FOREIGN KEY MEMBERS(MEMBER_ID)​ 외래키 설정이 잘못되었다.

BOARD_REG_DATE DATE DEFAULT '2023-03-22' 
=> 현재 시간이 아닌 DATE 타입에 문자데이터 '2023-03-22'를 기본값으로
지정했으므로 데이터 타입도 정확하게 일치하지 않고 현재 날짜 데이터도 아니다
